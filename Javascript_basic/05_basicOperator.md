# 기본 연산자

`+` : 더하기

`-` : 뺴기

`*` : 곱하가

`/` : 나누기

`%` : 나머지

위의 연산자들은 계산을 하기 위한 연산자이다.

<br />

## %(나머지) 연산자의 쓰임

1. 홀수와 짝수 구별

```javascript
x % 2 = 1;  // 2로 나누었을 때, 나머지가 1인 수는 모두 홀수
y & 2 = 0;  // 2로 나누었을 때, 나머지가 0인 수는 모두 짝수
```

2. 결과 값이 특정 값 이상이 되면 안되는 경우

```javascript
x % 5 = 0;  // 0 ~ 4 사이의 값만 변환
```

`5(특정 값)` 이상이 되면 안되는 경우에 결과 값에 `5`를 나누면 `5` 이하의 값만 반환된다.

<br />

## 거듭제곱

```javascript
const num = 2 ** 3; // 2의 3제곱
console.log(num); // reuslt => 8
```

거듭제곱을 계산하고 싶으면 `*(곱하기)`를 두번 적어주면 된다.

<br />

## 우선순위

수학과 같이 연산자는 우선순위를 갖는다.

`*(곱하기)`과 `/(나누기)`는 `+(더하기)`와 `-(빼기)`보다 우선순위가 높기 때문에, 원하는 값을 얻기 위해서 `()괄호`를 적절히 사용해야 한다.

<br />

## 연산자 줄여서 쓰기

```javascript
let number = 10;
//number = number + 5;
number += 5; // 위 코드와 똑같이 동작한다.

console.log(number); // result => 15

number -= 5; // number = number - 5 와 같다.
number *= 5; // number = number * 5 와 같다.
number %= 5; // number = number % 5 와 같다.
```

`number = number + 5`와 같이 `number`에 `5`를 더하고, 그 결과를 다시 `number`에 할당해주는 연산을 `number += 5`와 깉이 줄여서 사용할 수 있다.

`+=` 외에 `-=`, `*=`, `%=`도 줄여서 사용이 가능하다.

<br />

## 증감 연산자

```javascript
let a = 10;
let b = 10;
a++; // 값을 1만큼 증가
b--; // 값을 1만큼 감소

console.log(a); // result => 11
console.log(b); // result => 9
```

증감 연산자는 값을 1만큼 증가 혹은 감소시킨다.

증가 연산자는 `++`, 감소 연산자는 `--`로 `+`와 `-`를 두번 적어주면 된다.

### 증감 연산자의 순서

```javascript
let c = 10;
let d = 10;
let result1 = c++;
let result2 = ++d;

console.log(result1); // result => 10
console.log(result2); // result => 11
```

증가 연산자를 뒤에 넣으면 증가시키기 전에 값을 할당하고, 증가 연산자를 앞에 넣으면 증가시킨 후에 값을 할당한다.
